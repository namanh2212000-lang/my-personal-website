Đang tải thống kê...

📖 Hướng dẫn sử dụng:

  1. Thêm bài viết: Sử dụng form ở trên để thêm bài viết mới
  2. Download JSON: Click "Download JSON" để tải file posts.json
  3. Upload lên server:
    • Upload file posts.json vào thư mục /var/www/website/data/
    • Hoặc copy nội dung vào file data/posts.json trong VS Code
  4. Deploy: Git push và chạy update script trên VPS
