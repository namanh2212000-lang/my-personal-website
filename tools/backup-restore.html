📖 Hướng dẫn sử dụng

  1. Thêm bài viết: Sử dụng form "Thêm bài viết mới" ở trên
  2. Backup: Click "Tải xuống JSON" để backup dữ liệu về máy
  3. Push lên Git: Sau khi hoàn tất, chạy lệnh git push
  4. Test trên VPS: Truy cập website để test
  5. Restore: Khi cần, sử dụng "Tải lên JSON" để khôi phục
